Cuerpo Principal del Contenido

La etiqueta main es en HTML 5 una etiqueta semantica. Esto quiere decir que tiene que ver con el significado de lo contenido dentro de ella. Define el contenido principal de la pagina. Unicamente puede haber una sola etiqueta como esta en todo el documento.
El contenido escrito aqui dentro no se repite en el resto del sitio. Es usado por buscadores para indexar.
Conviene usar aqui textos que ayuden a encontra la pagina luego de procesos de indexacion de los buscadores.

Seccion etiquetas semanticas estructurales mas importantes

  • HEADER
  • FOOTER
  • SECTION
  • ARTICLE
  • NAV
  • ADRESS

Las etiquetas semanticas estructurales estan relacionadas directamente con el significado del contenido y no con los estilos ni las posicion dentro del documento.

Reglas para etiquetas semanticas:

  • Los distintos tipos de contenido se pueden asociar a las etiquetas SECTION.
  • Las unidades de contenido se asocian a las etiquetas ARTICLE. O sea que un SECTION anida multiples ARTICLES.
  • Las etiquetas HEADER Y FOOTER pueden ser anidadas dentro de otras HEADER o FOOTER. Osea un FOOTER podria anidar otro HEADER y FOOTER.
  • Las etiquetas NAV debe ser unica en el documento y corresponde a los elementos de navegacion del documento.
  • Reservar etiquetas h1 para usar dentro de etiqueras top level como main, header o footer.
